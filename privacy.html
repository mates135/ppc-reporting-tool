Privacy Policy

Last updated: July 2026

This privacy policy applies to the Struška PPC Reporting Tool ("the tool"), an internal application developed and operated by Martin Struška, an independent PPC/SEM specialist based in the Czech Republic ("the developer", "I", "we").

Data accessed

The tool connects to the Google Ads API using OAuth 2.0 authorization granted by the developer's own Google Ads Manager account (MCC). Through this connection, the tool reads the following types of data:

  • Account information (customer ID, account name, currency, time zone)
  • Campaign, ad group, and keyword data
  • Search term data
  • Performance metrics (impressions, clicks, cost, conversions, and related statistics)

The tool only reads data via read-only API calls. It does not create, modify, or delete any campaigns, ads, keywords, bids, or account settings via the API.

How data is used

Data retrieved via the Google Ads API is used solely to:

  • Generate internal performance reports
  • Support campaign analysis for the developer's own clients
  • Prepare summaries and materials shared directly with the relevant client whose account the data belongs to

Data sharing

Data is not sold, rented, or shared with any third party. Data belonging to a given client's Google Ads account is only used in connection with reporting for that same client. No aggregated or cross-client data is shared externally.

Data storage and security

OAuth credentials and any retrieved data are stored securely and are accessible only to the developer. Access tokens are not shared with any external party. Data is retained only as long as necessary to provide reporting to the relevant client.

Data retention and deletion

Retrieved reporting data is kept only as long as needed for ongoing client reporting. Clients or account owners may request deletion of any data related to their account by contacting the developer at the email address below.

Changes to this policy

This privacy policy may be updated from time to time. Any changes will be posted on this page with an updated revision date.

Contact

Questions about this privacy policy or how Google Ads data is handled can be sent to: struska.ppc@gmail.com
