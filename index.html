Struška PPC Reporting Tool

Developer: Martin Struška — independent PPC/SEM specialist (Czech Republic)

Struška PPC Reporting Tool is an internal reporting and analysis application built by Martin Struška, an independent PPC/SEM specialist. It is used exclusively by the developer to manage and report on Google Ads campaign performance for his own existing clients.

What the tool does

The tool connects to the Google Ads API to retrieve read-only performance data — campaigns, ad groups, keywords, search terms, and core metrics (impressions, clicks, cost, conversions) — across client accounts managed under a single Google Ads Manager account (MCC). This data is used to build performance reports and support campaign analysis for clients.

The tool does not perform automated changes to bids, budgets, or campaign settings. All optimization decisions are made manually by the developer in the Google Ads interface.

Who uses this tool

This is an internal-use tool. It is used only by Martin Struška to service his own clients — it is not distributed to or used by any external party.
